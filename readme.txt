== Description ==

Permanently remove all user-generated details of your Jigoshop store.

For more information visit: http://www.visser.com.au/jigoshop/

== Installation ==

1. Upload the folder 'jigoshop-store-toolkit' to the '/wp-content/plugins/' directory
2. Activate 'Jigoshop - Store Toolkit' through the 'Plugins' menu in WordPress

That's it!

== Usage ==

1. Open Tools > Store Toolkit
2. Select which Jigoshop details you would like to remove and click Nuke

Done!
